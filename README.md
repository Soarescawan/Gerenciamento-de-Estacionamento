#  Gerenciamento de Estacionamento

Sistema de **gerenciamento de estacionamento** desenvolvido em **linguagem C**, como trabalho acadêmico da disciplina de Programação de Computadores.

O projeto aplica conceitos fundamentais de programação estruturada, manipulação de memória, listas encadeadas, arquivos e modularização de código.

---

##  Funcionalidades

- Cadastro de usuários
- Cadastro de veículos
- Associação veículo ↔ vaga
- Validação de placas e vagas
- Remoção de espaços em entradas
- Leitura e escrita em arquivos `.txt`
- Busca e verificação de dados já cadastrados

---

##  Tecnologias Utilizadas

- Linguagem C
- GCC
- Arquivos `.txt` como persistência de dados
- Linux
